# Bigk-Grills

This is a restuarant management system for the Big k grills restuarant.

### Technologies

- Reactjs
- Context Api and Mobx
- Tailwind CSS

### Installation and Usage

Please ensure you have Nodejs (v8+) installed. Install the required dependencies using yarn

```bash
$ yarn
```

Run locally using

```bash
$ yarn start
```
